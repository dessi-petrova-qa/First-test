# **UI Automation Tests — Playwright + C#**
Automation UI test framework built with **Playwright**, **C#**, and **xUnit**, following Page Object Model (POM) architecture.  
The framework is designed for scalability, clean structure, and fast execution in CI/CD pipelines.

---

## 🚀 **Technologies Used**
- **C# / .NET 8**
- **Playwright 1.x**
- **xUnit**
- **Page Object Model (POM)**
- **GitHub**
- *(Optional)* GitHub Actions for CI
